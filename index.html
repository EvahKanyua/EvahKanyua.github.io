Python Projects

Automatic File Sorter: This Python program automates the sorting of files located in a specified directory by categorizing them according to their file extensions. It organizes the files into separate folders, each corresponding to a specific file extension. Upon execution, the program scans the designated directory, identifies the file extensions, creates folders for each unique extension, and moves the respective files into their corresponding folders.

Creating a BMI Calculator:This Python program assists in calculating the Body Mass Index (BMI) based on a person's weight and height inputs. By analyzing the BMI result, it categorizes individuals into specific weight categories, such as underweight, normal weight, overweight, or obese. This helps individuals gain insight into their weight status and promotes awareness of their health.

Delivery Price & Location Checker: This Python program simplifies the calculation of delivery costs for products by specifying the destination location. It enables users to input the name of the location for delivery, and based on predefined rates for each location, it computes the associated delivery cost. This tool streamlines the process of estimating delivery expenses for different destinations, facilitating efficient budgeting and logistical planning

Web Page Exploration: This Python program utilizes the BeautifulSoup library to scrape data from Wikipedia, specifically focusing on extracting information about the top-ranking high revenue companies in the USA. After retrieving the relevant data, the program performs cleaning and formatting tasks to ensure its usability. Subsequently, the cleaned data is organized and stored in a CSV (Comma-Separated Values) file format, enabling easy access and analysis. This automated process enhances efficiency in extracting and recording valuable information from websites for further utilization
